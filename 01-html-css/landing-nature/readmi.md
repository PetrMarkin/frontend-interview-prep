# Figma Landing Page

A static landing page built strictly with **HTML** and **CSS**, based on a provided Figma design. The layout is non-responsive (fixed-width) and follows the original pixel-perfect desktop design.

## 🎯 Project Overview

This project is a faithful implementation of a Figma mockup, focusing on pixel-perfect and visual accuracy. It uses only vanilla HTML and CSS, with no frameworks or JavaScript.

## 🧱 Page Structure

The page structure includes:
- **Header**
- **Main** (4 sections)
- **Footer**
